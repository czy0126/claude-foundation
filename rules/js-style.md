# JavaScript/TypeScript 代码风格规则

## 格式
- 2 空格缩进，100 字符行宽
- 使用 ESLint + Prettier，不偏离默认配置
- 导入顺序：第三方 → 本地模块（`@/` 别名）

## 类型（TypeScript）
- 禁止 `any`，必要时用 `unknown` + 类型守卫
- 所有公开函数必须有显式返回类型
- 优先使用 `interface` 而非 `type`（扩展性更好）

## 简单优先
- 一个函数做一件事，纯函数优先
- 不引入没人要求的抽象层
- 用可选链 `?.` 和空值合并 `??` 简化 null 检查
