# 安全规则

## 认证与密钥
- 绝不硬编码密钥、token、密码
- 密钥只通过环境变量或密钥管理器获取
- API 密钥不在日志、错误信息、git 提交中暴露

## 输入验证
- 所有外部输入必须验证后再使用
- SQL 查询使用参数化（禁止字符串拼接）
- 文件路径必须规范化并限制在允许目录内

## 敏感操作
- 用户数据变更必须记录审计日志
- 权限变更需要二次确认
- 生产环境操作需要显式的 `--production` 标志
