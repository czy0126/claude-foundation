# Python 代码风格规则

## 格式
- 4 空格缩进，120 字符行宽
- 导入顺序：标准库 → 第三方 → 本地模块，每组之间空一行
- 函数和类之间空两行

## 类型
- 所有公开函数必须有类型注解
- 使用 `| None` 而非 `Optional[...]`（Python 3.10+）
- 禁止裸 `except:`，必须指定异常类型

## 错误处理
- 不吞异常——要么处理，要么重新抛出
- 库代码优先返回 `Result[T]` 而非 `raise`
- 永远不要 `pass` 在 `except:` 块中不加日志

## 简单优先
- 一个函数做一件事，不超过 50 行
- 不用类实现可以写成函数的逻辑
- 不加没人要求的抽象层
